RiskCalculationService.php : 12

Processes raw Open-Meteo weather data into daily risk records and weekly summaries.

Risk values are computed from hourly precipitation probability and relative humidity using a weighted formula. Days with a risk value at or above RISK_THRESHOLD are flagged as flood risk days.

processDailyRecords()

RiskCalculationService.php : 60

Process raw Open-Meteo API data into a flat array of daily weather and risk records.

public processDailyRecords(array<string|int, mixed> $raw, int $days) : array<int, array<string|int, mixed>>

For each day in the forecast window, extracts daily min/max temperature and precipitation from the "daily" block, then scans the "hourly" block to collect all hourly humidity and precipitation probability values that fall on that day. Hourly values are averaged and fed into calculateRisk() to produce a risk score. If either averaged value is null (no hourly data for that day), the risk value and isRisk flag are null and false respectively.

Expected structure of $raw (as returned by WeatherService::fetchForecast()):

[
  'daily' => [
    'time'                  => string[],   // date strings e.g. "2025-06-01"
    'temperature_2m_min'    => float[],
    'temperature_2m_max'    => float[],
    'precipitation_sum'     => float[],
  ],
  'hourly' => [
    'time'                        => string[],   // datetime strings e.g. "2025-06-01T13:00"
    'relative_humidity_2m'        => float[],
    'precipitation_probability'   => float[],
  ],
]

Each returned record contains: 'date', 'minTemp', 'maxTemp', 'avgTemp', 'humidity', 'rainChance', 'rainMm', 'riskValue' (int|null), 'isRisk' (bool).

Parameters

Raw API response from Open-Meteo.

$days : int

Maximum number of days to process (capped at available data length).

Return values
array<int, array<string|int, mixed>> —

Flat array of daily records indexed from 0.

calculateRisk()

Compute a flood risk score from average rain chance and humidity.

private calculateRisk(float|null $rainChance, float|null $humidity) : int|null

Formula: risk = (rainChance × 0.5) + (humidity × 0.3), rounded to nearest integer. The result is an integer in the range 0–100 (uncapped — extreme inputs could theoretically exceed 100 if both inputs are at their maximum). Returns null if either input is null.

Parameters

Average hourly precipitation probability (0–100).

$humidity : float|null

Average hourly relative humidity (0–100).

Return values
int|null —

Computed risk score, or null if inputs are insufficient.
