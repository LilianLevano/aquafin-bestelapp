## Over het project

Dit project dient als een finale project voor een groep studenten voor de opleiding Graduaat programmeren aan de Erasmus Hogeschool Brussel.

Gevraagd aan ons is om twee applicaties te maken dat toelaat voor het opvragen van een overzicht van overstromingsrisico's en om aan techniekers van het bedrijf Aquafin toe te laten om op basis van prioriteiten van risicomaanden belangrijke tools en materialen te bestellen om deze te bestrijden.
